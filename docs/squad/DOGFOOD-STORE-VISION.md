# mcpwp.net as the omni-vertical dogfood store (Hadi, 2026-06-11)

> Captured mid-conversation. Status: vision, pre-spec. Decisions here are directional.

## The idea

mcpwp.net itself becomes the dogfood store — not a separate dropship site. It sells four
deliverable types, each one a live proof of a snapshot vertical, all operated by agents
through MCPWP, gated:

| Deliverable | Stack | Proves snapshot |
|------------|-------|-----------------|
| Merch — MCP / AI-as-harness / WP culture goods | Woo + print-on-demand dropship (no inventory) | #1 dropship |
| Courses — "MCPWP Academy" (AI ops on WP) | LearnPress (already installed on mcpwp.net) | #4 academy |
| Events — workshops, office hours, launches | events plugin + GHL calendars | #4 |
| Community subscription | membership plugin + Discord (T91/#421) | community leg |

LearnPress + WooCommerce are ALREADY active on mcpwp.net — the template pages drafted to
draft status 2026-06-11 (#runbook step C) regenerate on demand when the shop goes live.

## The culture thesis — brand: **Prefrontal Club** (Hadi-named 2026-06-11)

Merch margin irrelevant — merch = identity goods for a tribe with no flag yet: people who
love MCP + AI-as-harness + WordPress. The human as executive function over the agent fleet —
the prefrontal cortex of the operation. Community → E-E-A-T + distribution (AI-attention
thesis) → course buyers → addon BUILDERS → marketplace supply at 0% founding cut.
Flywheel: find plugin → join community → take course → wear shirt → build addon → sell on shelf.

## The adaptation factory (enabler)

The #507 rig doubles as a premium-plugin adaptation factory: install plugin locally →
agent introspects CPTs/options/REST → generate detection-gated MCPWP tool surface + tests
→ ship. Days per plugin. Seeds the marketplace shelves before third-party builders arrive.
First targets: events plugin + membership plugin (#508), B2BKing (B2B snapshot).

## Why it wins

Every sale on mcpwp.net is executed by agents through MCPWP, gated and audited — the store
IS the demo. STRATEGY §8.6 "bootstrap by being the first player" + §8.7 "one loop, one real
number, one real site" land on our own domain.
