Hey Product Hunt 👋

I'm Hadi, the founder of Mumega. We shipped MCPWP today and I'm excited to share what it does and why we built it.

**The problem:** WordPress has 43% of the web. AI agents can now do real work. But connecting the two requires custom code, REST API glue, or expensive SaaS platforms. We hit this wall every day as an AI agency.

**What MCPWP does:** Turns any WordPress site into an MCP server. Install the plugin, generate a key, paste one config block — your AI assistant (Claude, Gemini, GPT, Cursor, or anything MCP-compatible) can now manage your entire site.

250+ tools across 15 categories: Elementor page building, content CRUD, SEO audits, media uploads, menu management, WooCommerce, approvals with rollback, and persistent site memory.

**Why we think this is different from existing WP+AI tools:**
- Royal MCP has 37 tools. We have 250+.
- Elementor's Angie is cloud-only with credit limits. We're self-hosted.
- Agents can remember your brand voice across sessions (`wp_remember` / `wp_recall`).
- Human-in-the-loop approvals — stage any change, approve it, roll it back if needed.
- Role-scoped API keys — your content bot gets different access than your design bot.

**What I'd love your feedback on:**

1. **The install experience** — what tripped you up? We target < 5 min to first tool call.
2. **Which AI client are you using?** (Claude Desktop, Cursor, ChatGPT GPT, etc.)
3. **What would you build with this?** We've seen agencies use it for onboarding new sites, designers iterate Elementor layouts at speed, and marketers run SEO audits on 50-page sites in one prompt.

The plugin is open-source on GitHub ([Mumega-com/mcp-for-wp](https://github.com/Mumega-com/mcp-for-wp)). Pro plan adds the agency multi-site proxy, advanced Elementor tools, WooCommerce, and provider integrations.

Happy to answer anything — technical or otherwise. We're building in public and this is day one.

— Hadi
